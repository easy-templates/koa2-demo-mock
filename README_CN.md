## 介绍
这是一个基础仓库模板，用于快速创建空工程。

## 功能
本仓库为模板，创建基础文件，在此基础上进行相应修改。

## 快速上手

### 安装
```
$ git clone git@github.com:easy-templates/koa2-demo-mock.git
$ cd koa2-demo-mock
$ npm i
```

### 使用

* 将`koa2-demo-mock` 替换为对应工程名称
* 将`easy-templates` 替换为对应组织名称
