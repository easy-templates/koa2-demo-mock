## 介绍
这是一个基于koa2实现的mock服务。

## 功能

支持两种mock方式：
* json数据
* 代理
使用代理方式需要配合[koa2-demo-server](https://github.com/easy-templates/koa2-demo-server)

## 快速上手

### 安装
```
$ git clone git@github.com:easy-templates/koa2-demo-mock.git
$ cd koa2-demo-mock
$ npm i
```

### 使用

```
$ npm run start
```
